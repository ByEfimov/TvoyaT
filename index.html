Твоя траектория

О

проекте

"Твоя траектория" — это проект для учеников, которые не знают, кем хотят быть или нуждаются в поддержке своих идей. Наша цель — помочь ученикам с помощью использования метода индивидуального образовательного маршрута, а так же, научить их быть продуктивными без вреда для здоровья. Мы хотим доказать, что любой может найти себя и для этого совершенно не обязательно читать сотни статей. Все здесь — в Твоей траектории!
